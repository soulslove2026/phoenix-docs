# Phoenix Platform Documentation

Phoenix is an AI-native, globally scalable social platform centered on voice communities, digital identity, creator economy, real-time communication, digital gifts, agencies, moderation, analytics, and extensibility.

This repository is the canonical documentation source for the project. It is designed for both human engineering teams and AI coding agents.
