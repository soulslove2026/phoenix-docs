# Governance Checklist

- [ ] aligns with Phoenix Charter
- [ ] preserves user safety and trust
- [ ] has named owner
- [ ] identifies affected domains
- [ ] evaluates security and privacy
- [ ] evaluates data and AI impact
- [ ] evaluates cost and operations
- [ ] evaluates scalability and replaceability
- [ ] defines failure behavior
- [ ] defines rollback
- [ ] defines monitoring
- [ ] records unresolved risks
- [ ] answers: Does this make Phoenix stronger in ten years?
