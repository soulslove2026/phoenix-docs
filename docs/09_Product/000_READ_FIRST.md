## Foundation Position

Phoenix is a human-centered social platform designed around meaningful connection, live presence, trusted identity, safe participation, creator value, and carefully governed AI assistance.

The MVP must prove a coherent user loop rather than expose every planned capability. Product breadth is intentionally constrained so quality, safety, reliability, and learning remain strong.

## Product Gate

Implementation planning may begin only when each MVP capability has:
- a user problem and target segment;
- a journey and acceptance criteria;
- an owning bounded context;
- a data classification and contract;
- a security and abuse review;
- measurable outcomes;
- operational ownership;
- a rollout and rollback path.
