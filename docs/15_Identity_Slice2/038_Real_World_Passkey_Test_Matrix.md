# ID2-038 — Real-World Passkey Test Matrix

Record successful registration and authentication with user verification and discoverable credentials across approved browser, operating-system, platform-authenticator, and roaming-authenticator combinations. Never record credential IDs, session tokens, account identifiers, or user names.


Local real-device results are valuable engineering evidence but must use `environment: local`. Only staging or production Passkey records qualify to close the external-assurance gate.
