# ID2-037 — External Assurance Evidence Kit

Phoenix requires exactly one sanitized evidence record for each remaining Phase C gate. Passed records require redacted artifact hashes and approval metadata. Real evidence is forbidden from source control and must remain in approved encrypted storage.


Version 3.7.1 distinguishes structurally valid evidence from closure-qualified evidence. The assessment reports passed records from an unauthorized environment in `nonQualifyingKinds`.
