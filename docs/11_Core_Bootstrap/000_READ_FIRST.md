# Phoenix Core Bootstrap

This directory governs the first executable Phoenix repository and Slice 0 evidence.

## Bundle

- `phoenix-docs/`: updated documentation repository.
- `phoenix-core/`: a new standalone executable repository.

The bootstrap contains no product capability. It proves repository structure, buildability, configuration validation, health, readiness, structured logging, tests, CI, containerization, security baselines, and rollback.
