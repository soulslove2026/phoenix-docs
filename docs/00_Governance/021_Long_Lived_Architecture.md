# GOV-022 — Long-Lived Architecture

**Document ID:** GOV-022  
**Knowledge ID:** KN-GOV-022  
**Version:** 1.0  
**Status:** Ratified

Long-lived architecture changes without losing coherence. It requires modularity, clear ownership, replaceability, observability, resilience, versioned contracts, governed data, migration discipline, operational simplicity, and security by design.
