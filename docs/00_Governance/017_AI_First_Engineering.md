# GOV-018 — AI-First Engineering

**Document ID:** GOV-018  
**Knowledge ID:** KN-GOV-018  
**Version:** 1.0  
**Status:** Ratified

AI is a contributor, not an authority. Context precedes generation. Assumptions must be explicit. High-impact output requires human review. Tests, evidence, versioning, and privacy controls are mandatory.

AI may not silently invent financial rules, legal requirements, permissions, retention periods, moderation sanctions, data ownership, or API behavior.
