# GOV-021 — Tradeoff Framework

**Document ID:** GOV-021  
**Knowledge ID:** KN-GOV-021  
**Version:** 1.0  
**Status:** Approved

Evaluate major decisions across user value, safety, compliance, security, privacy, correctness, longevity, scalability, replaceability, maintainability, observability, operational simplicity, cost, delivery time, and reversibility.

A proposal must state what it optimizes and what it sacrifices.
