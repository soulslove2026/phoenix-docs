# PEH-012 — Engineering Checklists

**Document ID:** PEH-012  
**Knowledge ID:** KN-ENG-012  
**Version:** 1.0  
**Status:** Approved Baseline

## Before Coding

- [ ] problem and user outcome defined
- [ ] owner and domain identified
- [ ] acceptance criteria approved
- [ ] security and privacy reviewed
- [ ] data events defined
- [ ] failure behavior known

## Before Commit

- [ ] diff self-reviewed
- [ ] no secrets or personal data
- [ ] tests pass
- [ ] documentation updated
- [ ] migration impact reviewed

## Before Merge

- [ ] required reviews complete
- [ ] checks pass
- [ ] permissions tested
- [ ] rollback known
- [ ] telemetry present

## Before Release

- [ ] staged rollout plan
- [ ] dashboards and alerts ready
- [ ] release notes complete
- [ ] feature flags configured
- [ ] recovery path tested
