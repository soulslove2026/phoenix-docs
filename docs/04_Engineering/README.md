## Authority

These documents inherit authority from the Phoenix Constitution. Where a specialized engineering document conflicts with the Constitution, the Constitution prevails.

## Scope

The handbook governs:

- architecture;
- backend;
- Flutter;
- web and admin;
- APIs;
- databases;
- testing;
- security;
- observability;
- infrastructure;
- AI-assisted development;
- release engineering.

## Rule of Use

No production implementation should begin until the relevant engineering standards and acceptance criteria are available and approved.
