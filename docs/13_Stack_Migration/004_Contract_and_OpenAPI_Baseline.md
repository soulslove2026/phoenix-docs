# MIG-004 — Contract and OpenAPI Baseline

System endpoints use JSON Schema response contracts. Fastify generates an OpenAPI document and serves controlled documentation. Future APIs must define request and response schemas, stable error codes, ownership, and compatibility rules before implementation.
